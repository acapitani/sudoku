# sudoku
simple C sudoku solver

## Build with Visual Studio 2017 C++

## Execute

from command line:

sudoku hard.txt

sample input:

```
|---|---|---
|..3|.2.|6..
|9..|3.5|..1
|..1|8.6|4..
|---|---|---
|..8|1.2|9..
|7..|...|..8
|..6|7.8|2..
|---|---|---
|..2|6.9|5..
|8..|2.3|..9
|..5|.1.|3..
|---|---|---
```

or simply:

```
003020600
900305001
001806400
008102900
700000008
006708200
002609500
800203009
005010300
```
